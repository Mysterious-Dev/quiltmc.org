The Curseforge App allows you to download and manage mods from Curseforge in a dedicated app. To set up a Quilt profile, simply follow these steps:
1. Download and install the Curseforge App from [their website](https://www.curseforge.com/download/app)
2. Open the app and choose "Minecraft" from the list of games
3. Click "Create Custom Profile" in the top-right
4. Give the profile a Name
5. Pick Quilt as the Game Type
6. Click "Create"
